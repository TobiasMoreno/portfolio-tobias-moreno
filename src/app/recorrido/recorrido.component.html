Mi Recorrido

SQL

A principios de 2023 di mis primeros pasos en programación creando mi primera base de datos relacional con tablas de clientes y órdenes, y dominando consultas SELECT, JOIN y funciones de agregación.

C#

Durante el primer semestre de 2023 profundicé en POO con C#, construyendo una aplicación de consola para gestionar tareas y sentando las bases de Abstracción, Polimorfismo, Encapsulamiento y Acoplamiento.

Java

Entre marzo y agosto de 2023 me sumergí en Java, aplicando los conceptos de POO para desarrollar un sistema de gestión de biblioteca como proyecto de práctica.

Spring Boot

De septiembre a diciembre de 2023 aprendí Spring Boot: creé un API REST de gestión de empleados usando Spring Data JPA, Maven y patrones de diseño como DTO y repositorios.

Desarrollo Web

En enero–febrero de 2023 tuve mi primer contacto con HTML, CSS y JavaScript, creando landing pages y formularios de contacto para mis proyectos iniciales.

Angular

A principios de 2024 me adentré en Angular 19, construyendo aplicaciones dinámicas con componentes, servicios, rutas y consumo de APIs, y explorando librerías como Angular Material y RxJS.

Netlify & DevOps

Aprendí a desplegar mis apps con GitHub Actions y Netlify, configurando CI/CD, lo que reforzó mis habilidades en automatización y despliegue continuo.

Techforb – DevOps

Desde Marzo de 2025 formo parte de Techforb como desarrollador, donde trabajo en la implementación de funcionalidades y mejoras de rendimiento usando Angular y Spring Boot. Colaboro con el equipo en el diseño de arquitecturas limpias, escribo pruebas unitarias y promuevo buenas prácticas de código.

Techforb – Creditú

Desde agosto de 2025 formo parte del equipo de Creditú, un SaaS de créditos hipotecarios para el mercado chileno.
Más allá del desarrollo en Vue y NestJS, mi foco estuvo en mejorar la arquitectura y la performance del sistema, optimizando integraciones externas y logrando una reducción del 45% en tiempos de respuesta.
Participé en decisiones de despliegue y arquitectura en AWS (Lambda, S3, base de datos), asegurando que la plataforma sea escalable, mantenible y preparada para crecer.
